Claude Code 专区

功能发布：v2.1.129 5/6 凌晨

5/6 01:40 UTC 发布，核心变更：

  • --plugin-url <url> 新 flag：从 URL 直接拉取 .zip plugin archive 用于当前 session（无需提前安装）
  • CLAUDE_CODE_PACKAGE_MANAGER_AUTO_UPDATE：Homebrew / WinGet 安装时后台自动 upgrade + 提示重启
  • CLAUDE_CODE_FORCE_SYNC_OUTPUT=1：强制同步输出（解决 Emacs eat 等终端漏检问题）
  • DISABLE_UPDATES env var：完全封锁所有更新路径（比 DISABLE_AUTOUPDATER 更严格）
  • Gateway /v1/models discovery 改为 opt-in（CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY=1），此前 v2.1.126–128 为自动；claude plugin tag 新命令
  • skillOverrides 正式生效：off / user-invocable-only / name-only 三级控制 skill 可见性
  • Bug 修复：/context 浪费 1.6K tokens、/clear 未重置 terminal tab title、睡眠唤醒 OAuth 竞态登出、1 小时 prompt cache TTL 被降级为 5 分钟等
