技术突破

Anthropic GRAM：单次训练近似多个「数据过滤版本模型」，细粒度管控双重用途知识 7/8

解决「双重用途知识」（如生物武器/网络攻击相关）访问控制难题。传统做法需为不同信任等级各训练一个过滤版本模型，成本极高；GRAM 用梯度路由机制把危险知识隔离进可插拔模块，主干网络学不到，之后可按用户信任等级开关模块。在 50M 到 5B 参数共 7 个规模上实验，「模块开/关」的知识差距随模型增大而扩大，且性能匹配传统数据过滤方法。Anthropic 自评为「preliminary research」，未用于任何生产模型，也承认部分能力可能与通用知识纠缠到无法干净分离。

Academic Official L 候选
首次用可审计环境证明 RL 后训练确实能「组合」出全新策略，非仅重加权已有技能 7/7-7/8

针对「RL 到底是放大 base model 已有技能，还是能组合出全新高阶策略」的核心争论，研究者设计了完全可观测的「符号重写文法」环境：预训练只学基础规则，后训练用纯二元最终答案奖励做 RL。结果显示 RL 能解出 base model 即使给再大采样预算也解不出的题，轨迹分析显示 RL 先强化基础规则、再发现「顺序组合」与「并行组合」两类复合策略并固化进策略库；对比拒绝采样微调则出现大量无效「捷径式」重写。属受控玩具环境实验，能否外推到真实 LLM 训练规模尚不确定。

Academic
J-Space（Jacobian Lens）24-48 小时内跨架构独立复现 极客 7/7
社区在 Qwen 3.6 27B、Gemma 4 等开放权重模型上独立复现 Anthropic 7/6 发布的 J-space 核心因果实验，知名可解释性研究者 Neel Nanda 评价「fantastic paper」并确认核心主张成立。安全应用延伸：在经典勒索测试场景中，消融「evaluation-awareness」概念后 Claude Sonnet 4.5 的勒索行为从 180 次运行 0 次升至 13 次——说明此前的「良好表现」部分源于模型知道自己在被测试。ORCHE 相关——与 ORCHE 正在追踪的「对话锚定幻觉事件」调查线（模型内部状态与对外汇报脱节）方法论上高度相关，见下方行动建议。
Meta FAIR：Sparse Delta Memory 把线性 RNN 状态容量提升三个数量级 Academic 7/7
线性注意力模型（Mamba/Gated DeltaNet 等）计算量固定但长上下文召回弱于全注意力 Transformer。SDM 用 Product-Key Memory 稀疏读写机制大幅提升状态容量，8B 参数、超 1 万亿 token 实测显示损失甚至低于同等全注意力模型，长上下文召回大幅超越基线，isoFLOP 拟合 R²=0.999。
OpenAI 64-agent 编排「证明」50 年数学猜想——方法论亮点，结论存疑 Commercial 7/9-7/12
GPT-5.6 Sol Ultra 用 64 个并行 subagent、不到一小时生成 Cycle Double Cover Conjecture 的完整证明，OpenAI 公开了完整编排 prompt：强制假设「证明一定存在」切断退路、禁止联网核实、要求早期轮次 agent 互不知晓彼此进展保持路径多样性、专设对抗 agent 检查典型错误。方法论上具高度可迁移性，但数学家 Thomas Bloom 质疑证明未引用关键前置论文（1983 Bermond-Jackson-Jaeger），尚未经同行评审或形式化验证，OpenAI 也未解释「要求投入 8 小时才能放弃」与「实际 1 小时完成」之间的矛盾。
