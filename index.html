技术突破

MEMENTO: LLM 自主管理推理 context 4/10

微软研究院开源论文 + vLLM 扩展。训练模型将推理过程切分为「块」，每块压缩为 memento（密集语义摘要），推理时只 attend mementos 而非保留原始 token——实现 KV cache 物理压缩而非 masking。

单卡 B200 + Qwen3-8B 实测：peak KV cache 降 2.5x，吞吐量升 1.75x。vLLM overlay 已可用（非底层 kernel 改动），公开 OpenMementos 数据集（22.8 万条 reasoning trace）。arXiv:2604.09852 / 项目页

对 ORCHE 意义：Phase 3 memory 架构的关键参考——长 session 推理的 context 管理，与「有状态运作」直接相关。

四层 Agent Memory 架构：Knowledge / Memory / Wisdom / Intelligence 4/13 Academic L 候选
arXiv:2604.11364。诊断 CoALA / JEPA 缺少「Knowledge 层」独立持久性语义——知识（indefinite supersession）、记忆（Ebbinghaus decay）、智慧（evidence-gated revision）、推理（ephemeral）混在一起导致结构性错误。提供 Python+Rust 参考实现。本周同期 memory 论文集群 5+ 篇，社区痛点聚焦。ORCHE 现有 memory 设计正对应 Knowledge 层。
Pando: Interpretability 工具在模型不自我解释时几乎全失效 4/13 Academic
CMU 训练 720 个模型分离「启发增益」vs「白盒内部信号」混淆（elicitation confounder）。结论：模型不解释时 SAE、circuit tracing、logit lens 几乎无效；gradient-based attribution 是唯一可靠（+3-5pp），relevance patching 最优。推翻 SAE 在 alignment auditing 的普遍适用假设——对「AI 内在 alignment」可验证性有理论挑战。
AAR benchmark：agent 瓶颈在导航不在工具调用 4/17 Academic
arXiv:2604.10261。发现现有 6 大 tool-use benchmark 中 55-100% 是简单 2-5 步线性链。构建含分叉-合并工具链的 DAG benchmark（1400 实例），最强 agent 仅 37.2% 准确率，错误以导航失败为主（27-52%），tool call 错误 <17%。论文明确提及 Claude Code 以 6x 更少 token 匹配 Codex CLI 37% 准确率。
