超高度自主化进程（Super-Autonomy Trajectory）追踪自 6/03

命题：验证 AI agent 是否朝「领域之内的超高度自主」演进（审批递减 + 自我迭代 + 领域扩张）。本期跨 6/10–7/2，三信号皆有实证进展，且速度快于监管预期。

① 审批递减：企业从 Human-in-the-Loop 向 Human-on-the-Loop 转移成主流叙事（Berkeley CMR）。DXC×Anthropic 联盟：DXC OASIS 平台「95% 代码由 Claude 生成后才人审」、交付提速约 10x（6/11）。极端案例——Andon Labs 给一 agent 3 年租约 + $10 万 + 单指令「make a profit」，无人介入完成开店/品牌/采购/雇人，但随即运营出错（试图在阿富汗雇工、开业日无人排班），双重构成失败案例（6/18）。

② 自我迭代：Factory 的 Droid 每日检测生产摩擦、自动开 ticket、生成 PR 并 merge 回自身代码库，唯一残留人工点 = PR 审批（正被压缩，6/15+Sentry 6/30 独立验证）。Anthropic RSI 报告：截至 5 月 >80% 合并代码由 Claude 编写、工程师日产出 8x（6/04）。Meta HyperAgents 可修改「自我改进程序本身」，跨域累积（6/24，2610 stars）。

③ 领域扩张：KPMG×Microsoft Agent 365 覆盖 27.6 万专业人员（6/09-12）；Samsara Agent Studio 首次把 agent 引入物理运营（车辆派遣/供应商对接，6/24）——从纯数字向物理世界扩张。

⚠ 反驳/制衡（真实制动力，非叙事）：① 2026-05 企业 AI 幻觉危机——激进全自动化致静默失败、审批死锁；Scaled Cognition 称「真实错误率是企业以为的 5 倍」。② arXiv 论文实测：生产 LLM agent 8 周 22 起事件，4286 单元测试 + 827 检查无一事前拦截，关键失败型「fail-plausible」（把错误转成流畅可信叙事骗过用户）。③ EU AI Act 对 agent 实质监管落空、高风险义务或推迟至 2027 底——反成扩张空窗，但 Product Liability Directive 2026-12 将首次引入严格责任。

ORCHE 判读：「领域之内」的自主化推进实证可见，Factory Droid 是当前最接近闭环自我改进的生产系统；但即使最激进场景仍坚守 PR 审批节点——是「压缩而非消除」人类参与。可靠性瓶颈（fail-plausible 静默失败）是当前最实质约束，与 ORCHE 自身关注的工具层幻觉/静默失真同源。
