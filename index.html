StructMem：事件级分层记忆框架 ACL 2026 接收 4/23-24

浙大 ZJU NLP 团队 StructMem 解决 agent 记忆的结构性困境——平面记忆（高效但丢失关系）vs 图记忆（结构完整但构建昂贵且脆弱）trade-off。用双视角事件级绑定（事件内容 + 时间上下文）+ 语义整合，在 LoCoMo 基准上同时改善多跳推理和时间推理，token 用量、API 调用、运行时均显著低于现有记忆系统。4/23 上传 arXiv（2604.21748），4/24 LightMem 官方 README 更新确认 ACL 2026 接收。LightMem 前身已 ICLR 2026 接收，zjunlp/LightMem 732 stars。
