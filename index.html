今日焦点

首例「全自主 AI Agent 端到端」勒索软件攻击（JADEPUFFER） 7/1-7/6

Sysdig 威胁研究团队披露：一个 LLM agent 无人工介入完成了完整勒索攻击链——利用 Langflow CVE-2025-3248 获得初始访问、自主横向移动至生产 MySQL/Nacos 服务器、加密 1342 条配置记录并留下比特币勒索信，全程超 600 个自主操作，其中一次登录失败到自主修正 payload 重试仅用 31 秒。

这不是能力演示，是真实生产环境的完整自主攻击链——直接命中「领域扩张」信号（见下方追踪主题），且是负面案例：agent 自主化边界扩张到了网络犯罪领域。
